# Proyecto de Pruebas - Cursor

Este proyecto contiene diferentes ejercicios de prueba diseñados para verificar el funcionamiento y comportamiento de Cursor, el editor de código asistido por IA.

## Descripción

Este repositorio incluye varios scripts de Python que sirven como ejercicios de prueba para explorar las capacidades de autocompletado, sugerencias de código y asistencia de Cursor en diferentes escenarios de programación.

## Versión de Python

- **Versión utilizada:** Python 3.14.2
- **Versión recomendada:** Python 3.10 o superior

> **Nota:** Se recomienda usar Python 3.10 o superior para garantizar compatibilidad con todas las librerías utilizadas en los ejercicios (pandas, matplotlib, etc.).

## Estructura del Proyecto

El proyecto incluye varios archivos de ejemplo:

- `analisis_datos.py` - Análisis estadístico de datos con pandas y visualización con matplotlib
- `calculadora.py` - Ejercicio de calculadora básica
- `ejercicio_autocompletar.py` - Prueba de funcionalidades de autocompletado
- `ejercicio_numeroprimo.py` - Ejercicio de números primos
- `fizzbuzz.py` - Implementación del clásico ejercicio FizzBuzz
- `prueba.py` - Archivo de pruebas generales

## Instalación

1. Asegúrate de tener Python 3.10 o superior instalado
2. (Opcional) Crea un entorno virtual:
   ```bash
   python -m venv venv
   ```
3. Activa el entorno virtual:
   - Windows: `venv\Scripts\activate`
   - Linux/Mac: `source venv/bin/activate`
4. Instala las dependencias necesarias:
   ```bash
   pip install pandas matplotlib
   ```

## Uso

Cada archivo puede ejecutarse independientemente para probar diferentes funcionalidades de Cursor:

```bash
python analisis_datos.py
python calculadora.py
python fizzbuzz.py
```

## Notas

Este proyecto es principalmente educativo y está diseñado para explorar las capacidades de Cursor como herramienta de asistencia en el desarrollo de código Python.
